что такое структуры данных?
это структуры,которые могут хранить данные .В Python есть четыре встроенные структру данных: список,кортеж,словарь
и множества

зачем они нужны?
они нужны чтобы хранить данные пользователя

что такое Hash Map?
это структура данных,которая подразумевает хранение данных в виде пар ключ/значение.


что такое LInked List?
это связной список,который прт удаление кого-то узла(обьекта) из списка освобожлает места и экономит память

в каких случиях используются Linked List и Hash Map
Linked List используется для экономии места при удаление узла
Hash Map используется для храниние данных в виде пары ключ-значение.
Он обычно используется для легкого и быстрого извлечения.
Данные Hashmap могут быть получены с помощью ключа.
